# AGENTS.md

## Reporting

- Production-facing files must not preserve chat framing or conversational scaffolding from the working thread.
- Do not write phrases such as:
  - "this version only covers three things"
  - "as discussed above"
  - "the point here is"
- Reports should be written as standalone artifacts:
  - state the structure
  - state the current condition
  - state the recommended next move
- Keep delivery documents readable outside the chat context.
